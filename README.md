## <img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="28"/> &nbsp;About Me

<br/>

<img align="right" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" width="340" />

I'm a **System Design–focused Flutter Engineer** who builds **scalable, resilient software systems** — across mobile, backend, and real-time infrastructure.

I don’t just build features. I design how systems behave under pressure, scale across users, and recover from failure.

My work spans:

- ⚙️ **Distributed systems thinking** — data flow, consistency, fault tolerance  
- 📡 **Real-time architectures** — event-driven, WebSockets, sync engines  
- 🧠 **Scalable app design** — Clean Architecture, modular systems  
- 🔋 **Offline-first systems** — sync strategies, caching, resilience  
- 🔄 **End-to-end system behavior** — from UI → network → backend → storage  

I approach every product like a system — not just an app.
